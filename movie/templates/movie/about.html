About Erudition

The Largest Web Developer Site on the Internet


What is Erudition?

Erudition is a web developers site, with tutorials and references on web development languages such as HTML, CSS, JavaScript, PHP, Python, DBMS, Java, C, C++,Django and DS, covering most aspects of web programming.

The site derives its name on the quote that is "The ability to show knowledge".

Erudition was originally created in 2020 by two students (Vinayak Srivastva and Anjali Verma) from United Institute of Manangement (FUGS), Prayagraj.

Easy Learning

Erudition has focus on simplicity.

Erudition provides questions hfor helping you in interviews and other.

Erudition practice easy and straight-forward learning.

Erudition uses simple code explanations with simple illustrations of how to use it.

Erudition start from basic level up to professional references.

Try It Yourself

Erudition presents thousands of code examples.

By using our online editor, Try it Yourself, you can edit examples and execute computer code experimentally, to see what works and what does not, before implementing it.

Erudition is Free

Erudition is, and will always be, a completely free developers resource.
