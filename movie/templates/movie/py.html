    • Introduction:-

    Python is a high-level,object oriented interpreted programming language.Python is always known as Scripting Launguage.
    OOPS(object oriented programming system) provided a real life approach(objects).Python follows the OOPS concept.It uses to make/develop application/software.Such as desktop application,mobile application,web application,embedded (Robotics) application.

    There are used interpreter to change the high-level language to machine-level language.It is also known as beginer level language because of its easier syntax.It is platform independent.It can extend that means make or add changes.It is used to make GUI application.We can also perform on database server,it is scalable across platform.Python is secure because in OOPS one object can not access the properties and functionality of another object.The file extension is .py. Such as python.py.

    Rules of writting source code :-

    1. Do not start with space whenever you start writting the code.
    2. Space refers to unexpected indent error.
    3. We can use space inside the code block.
    4. Always write new instruction within new line.
    5. Always write the instruction with begining.
    6. Whenever write multiple code/instruction then use semicolon (;) at the end of each instruction.

    OOPS concepts in Python:-

    1.Encapsulation:-Encapsulation is a process of wrapping up of data into a single unit. It provided the data security by the binding the properties and functionality.It is just like capsul in real life where multiple medicines are combine into a single unit.

    2.Inheritance:-Inheritance is a process of aquiring properties and functionality from parent class into child class.By the use of inheritance we can make reusable the code.It is just like a child inherited his/her parents properties.
    There are following type of Inheritance:-
  • 1.Single Inheritance
  • 2.Multiple Inheritance
  • 3.Multilevel Inheritance
  • 4.Hierarchical Inheritance
  • 5.Hybrid Inheritance

    • 3.Polymorphism:-Polymorphism refer to the one object take more than form.Just like a man who is son as well as father,brother etc.By the use of Polymorphism we can reduce code complexity.In python only operator overloading allowed,method overloading is not allowed because in python method can receive different type of parameter.

    4.Abstraction:-Abstraction is a process of hiding the unnecessary data from the user.In other word, we can say that by the use of Abstraction we can reduce the code complexity.The real life example of Abstraction is mobile phone which have only those functionality is outside which can be used by user.

    Lambda Expression

    Lambda Expression also known as one linear function or anonymous function.It is used to only one statement use like method.It is a standard definition method.Cann't write complex code.
    Syntax:-
         lambda var1,var2,var3,...: expression
    Example:-
          powfun=lambda a,b:(a+b)**2/2
          print(powfun(2,6))

    Calculate area of circle:-
          area=lambda r: 3.14*(r)**2
          r=input("Enter radius=")
          print(area(r))
