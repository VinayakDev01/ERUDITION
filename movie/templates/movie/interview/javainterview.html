Welcome!    java Interview Questions

1)What is Java?

Java is a fully OOPs language.After compilation it creates Byte code which is run on any platform that's why it is known as platform independent language. It is also known as 'WORA' i.e write once run anywhere.



2) What is Byte Code?

After compilation Java compiler converts Java program into byte codes which is executed by JVM(Java Virtual Machine).



3) What is JVM?

Java Virtual Machine is a software which is use to interpreted the byte codes.JVM does not compile byte code,it interprets byte code to execute java program.



4) Write about Java data types?

Data type is used to store user defined data.It defines the type of data to be stored that can be byte, short, long, integer, float, double, boolean, character datatype.



5) What is 'FOR'loop?

For' loop is a control flow statement which is used to repeat a block of codes until certain condition is met.At first it has a initial condition for staring of loop, second for testing condition to stoping the loop and third , an increment condition for incrementing the initial value.



6) What is 'while' loop?

The 'while' loop is used for repeating a block of codes or set of instructions.This loop repeats the statements until a condition becomes false.It means, it repeats the statement till a condition is true.The number of repetition is not fixed it is based only on the given condition.



7) What is do...while loop?
'do..while' loop is used to repeat the block of codes.The testing condition is given in 'while' part of the loop but no one condition is given in 'do' part of the loop. The 'do..while' executes the instructions with in the loop at least once, after that it checks the testing condition.If testing condition in 'while' part of the loop is true, then it executes the instructions again.When condition is false, then the loop terminates. The number of repetition is not fix in this loop.It depends on the condition which is given to the 'while' part of the loop.



8)Tell me something about 'conditional statement'.

A conditional statements lets choose which statement executed next.There are various types of conditional statement in Java. 1-if statement:It executes 'if' block when condition is true.
2-if-else statement:It executes the 'if' block when condition is true otherwise 'else' block is executed. 3-if-else-if statement:In This statement multiple statements executes one condition. 4-nested if statement:This statement represents 'if' block within another 'if' block.At first,when outer 'if' block condition is true then inner 'if' block condition executes .



9)What is switch statement??

A switch statement is a type of selection control mechanism it is used to allow the value of a variable to change the control flow of program execution.It gives a more expressive way to compare a value with multiple forms. The switch has more than one case blocks and an optional default case also.
