Welcome!    C++ Interview Questions

1)Describe something about C++ language?

C++ language is a partially OOPS language.It is based on class and Objects.This language is advanced version of C language.



2) Write basic concepts of OOPS?

Concepts of OOPS are given below:
1)Abstraction
2) Encapsulation
3) Polymorphism
4) Inheritance



3) What is a class?

A class is a blueprint that contains a similar type of objects in it. It describes all the details of an object.



4) What is an Object?

Objects have its own state, behavior and identity.It is an instance of a class.



5) What is an abstraction?

Abstraction means 'To Hide', that means abstraction is used to hide the unnecessary details , only required details for an objects are shown.



6) Do you know about encapsulation?

Wrapping up data in a single unit is known as encapsulation.



7) What is Polymorphism?

Polymorphism is more than one form.It assigns value in a subclass which was already declared in the main class.



8) What is inheritance?

Inheritance is used to inherit some properties from parent class to child class.Child class has its own properties also.



9)For what super keyword is use?

Super keyword invokes immediately the parent class method.If parent and child class have same name method and we want to call parent class method then we have to use Super keyword.







10) Explain the constructor.

Constructor initializes the object of its class having same name as the class.
i)A constructor must have no return type.
ii)Constructor name should not even void.



11) Define Destructor?

Destructor is a method.It is always automatically called when the object is need to be destroy.With the symbol '~ ' known as 'tilde',destructor name is also same as class name.



12) What is a virtual function?

With the use of keyword "virtual" during function declaration,virtual function is a member function of a class which is redefined by its derived class and declared within a base class. In C++ programming language ,virtual function can be declared using a token.Virtual function cannot be a friend function of a class.



13) What is a friend function?

A function that is allowed a friend of a class to access Public, private, or protected data in that same class is known as friend function.The function cannot be access that information if it is defined outside the class. There is no effect on friend function of access control keywords (like private, public, or protected) and it can be declared anywhere in the class declaration



14) What is a pure virtual function?

A function which can be overridden in the derived class but cannot be defined is known as pure virtual function.By using the operator =0 a virtual function can be declared as Pure.



eg.
Virtual void functionA() // Virtual, Not pure
Virtual void functionB() = 0 //Pure virtual



15) What is function overloading?

When more than one function have the same name with different parameters in a same class then it is called as function overloading. It can be perform different tasks.



16) What is the main difference between a class and an object?

Objects can hold various informations and it is an instance of a class.,but classes contains similar types of objects.



17) What is the main difference between overloading and overriding?

Overloading is static Binding,whereas Overriding is dynamic Binding.Overloading is nothing but the same method with different arguments,and it may or may not return the equal value in the same class itself.Overriding is the same method names with the same arguments and return types associated with the class and its child class.



18) What is method overriding?

When more than one method have the same name in different class i.e if a child class has same method as parent class then its known as method overriding.



19) What is the use of finalize method?

Finalize method is used to remove resources which are not currently used.This method is accessible by a derived class and it is protected.



20) What is an abstract class?

We cannot create an object in abstract class but it can be inherited.It can contain only an Abstract method.



21)What is Exception handling?

Exception handling is an event which is used to handle the run time errors caused by exception by using try catch, throw, throws, and finally keywords.



22) What are the access modifiers?

Access modifiers determine the scope of the method or variables that can be accessed from other various objects or classes.There are five types of access modifiers,and they are as follows:
1) Private
2) Protected
3) Public
4) Friend
5) Protected Friend



23)How many types of Arguments?

There are two types of Arguments:
1)Call by Value : Passed value can modifies only inside the function, and it returns the same value which is passed into the function.
2)Call by Reference :Passed value can modifies in both side(inside&outside)of the functions and returns the same or different value.



24) What is the default access modifier in a class?

Private access modifier



25) What are all the operators that cannot be overloaded?

The operators which cannot be overloaded are given below:
1) Scope Resolution (::)
2) Member Selection (.)
3) Member selection through a pointer to function (.*)
